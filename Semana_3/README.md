Conclusión:

"El poder de la Visualización de Datos con Matplotlib en el Aprendizaje Automático"

Este análisis exploratorio y visualización de datos con Matplotlib, es una herramienta muy  importante. Nos permite entender qué está pasando con los datos, identificar patrones relevantes, relaciones y valores atípicos de una forma más clara, que solo ver números y tablas.

Los gráficos nos permiten detectar si hay errores en los datos, como tambien ver si hay relación entre variables para detectar una correlación útil para poder predecir. 

Entiendo que el gran desafío sobre su uso, es la elección del gráfico, las escalas y sobre todo, que la interpretación de los mismos, no nos lleven a tomar decisiones equivocadas o comunicar resultados erróneos.
