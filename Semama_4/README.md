Clase 4: **Aprendizaje Supervizado**

Esta clase aborda los fundamentos del Aprendizaje Supervisado.

Se presentan dos modelos esenciales: Regresión Lineal y Regresión Logística, explicando cómo se utilizan para predecir valores y clasificar datos.

Además, se trabaja con la preparación, exploración y organización de datos, la creación de conjuntos de entrenamiento y prueba, y la implementación práctica de estos modelos en Python utilizando librerías como pandas,
NumPy y scikit‑learn.

La clase combina teoría y práctica para comprender cómo construir, evaluar y visualizar modelos predictivos.
